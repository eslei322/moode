$_select[ashufflesvc1]$_select[ashufflesvc0]
当列表中的最后一首歌曲回放完毕，“ 自动穿插 ” 会将音乐库中的两首随机选择的歌曲添加到回放列表的末尾，而且是个重复过程。
这样就可以连续回放而无需增加列表，并且具有两个音轨可以使用 MPD 交叉渐变。
备注：“ 自动穿插 ” 将替换 “ 随机回放 ” 按钮设定的功能。
$_select[autoplay1]$_select[autoplay0]
Moode 启动后自动回放对上一次关机前最后一个回放曲目。
$_select[rotaryenc1]$_select[rotaryenc0]
启用轮转编码设备以控制 MPD 音量。
驱动程序参数：
- poll_interval: 单位时间为毫秒的音量更新延迟在轮转轨迹的回应。 默认值 = 100
- accell_factor: 每一级音量 增/减 的数值是 1 （ 选择 “ 1 ” ）还是下一项设定的数值 （ 选择 “ 2 ” ）。 默认 = 2
- volume_step: 快速调节音量时每一级 增/减 的数值。 默认值 = 3.
- pin_a, pin_b: Broadcom SoC pin numbers. Default = 23, 24.
注意： 这个不是回放面板上的音量控制
$_select[usb_volknob1]$_select[usb_volknob0]
激活 “ triggerhappy key mapping daemon ” 控制 USB 音量。
了解更多： /etc/triggerhappy/triggers.d/media.conf for key/event mappings.
设置当点击音量显示轮转轨迹预前区域时音量增加值。
设定 MPD 最高音量。
以上两项应用于回放面板上的音量控制。
$_select[invert_polarity1]$_select[invert_polarity0]
反转输出音频相位 +/- 到 -/+.
计量单位是秒，设置为 0 以禁用交叉淡入淡出。
注意： MPD 使用其音频缓冲区进行交叉渐变，因此对于相应的音频格式，必须有足够的缓冲区空间来执行请求的淡入淡出秒数。
- 700 Hz, 4.5 dB - 默认
此设置在用耳机收听时最接近摹仿扬声器放置 3 米距离，方位角为 30 度。
- 700 Hz, 6.0 dB - Chu Moy (most popular)
此设置接近 Chu Moy 交叉叠加参数
- 650 Hz, 9.5 dB - Jan Meier
此设置接近 Jan Meier 的 CORDA 放大器中实现的交叉馈电的参数，使信号变化最小，可以轻松收听耳机。
$_select[mpd_httpd1]$_select[mpd_httpd0]
在指定端口上将 MPD 输出作为 HTTP 音乐媒体发送。
默认端口是 8000，端口范围是 8000-9000。
使用 FLAC 压缩级别 0 （ 最快，最小压缩 ） 的或 320 kbps 的 LAME （ MP3 ） 作为发送编码格式。

此均衡器使用 Tim Goetze 编写 “ CAPS 数字信号处理程序套件 ” 里面的 EqFA4p 组件，串联四个 Mitra-Regalia 峰值均衡器滤波器，和 Fons Adriaensens "Parametric1" 矢量运算参数型均衡器 [fafil] （ four-band LADSPA parametric EQ ） 差异很小。 串行滤波器并行化导致输出滞后三个样本。
